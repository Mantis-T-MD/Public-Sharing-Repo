Particle Equilibrium

Animated walkthrough — tap Next to advance

0 The Problem

Three forces act at a single point:

  • Cable A — up-left at 30° from horizontal
  • Cable B — up-right at 60° from horizontal
  • Weight — 500 N straight down

Find: Tension in Cable A (TA) and Cable B (TB)

1 IDENTIFY 8-STEP

All three forces meet at one point.

One point → Particle problem
No moments. Only ΣFx = 0 and ΣFy = 0.

This means we have 2 equations available.

2 FREE BODY DIAGRAM 8-STEP

Replace cables with tension forces. Keep the weight. That's your FBD:

3 forces, 2 unknowns (TA, TB). The 500 N is given.

3 COUNT 8-STEP

Unknowns vs Equations
Unknowns: TA, TB → 2
Equations: ΣFx=0, ΣFy=0 → 2
2 = 2 → Solvable ✓
If unknowns > equations → problem is indeterminate (can't solve with statics alone). Here we're good.

4 DECOMPOSE 8-STEP

Break each force into x and y components:

Sign rule: Right = +x, Left = −x, Up = +y, Down = −y
Cable A points left → its x-component is negative

5 MOMENT POINT — SKIP 8-STEP

Particle problem → all forces at one point → moment arm = 0 for everything.
ΣM = 0 gives us 0 = 0. Useless. Skip this step.

This is why particles can only solve 2 unknowns — you only get 2 equations, not 3.

6 WRITE EQUATIONS 8-STEP

→+ ΣFx = 0
−TAcos30° + TBcos60° = 0
Cable A pulls left (−), Cable B pulls right (+), weight has no x
↑+ ΣFy = 0
TAsin30° + TBsin60° − 500 = 0
Both cables pull up (+), weight pulls down (−)
Two equations, two unknowns. This is just algebra from here.

7 SOLVE 8-STEP

From ΣFx = 0 — isolate TA
TAcos30° = TBcos60°
TA = TB · cos60° / cos30°
TA = TB · (0.5 / 0.866)
TA = 0.5774 · TB
Substitute into ΣFy = 0
(0.5774 · TB)sin30° + TBsin60° = 500
0.5774 · 0.5 · TB + 0.866 · TB = 500
0.2887 · TB + 0.866 · TB = 500
1.1547 · TB = 500
TB = 433.0 N
Back-substitute for TA
TA = 0.5774 × 433.0
TA = 250.0 N

8 CHECK 8-STEP

Verify ΣFx = 0
−250·cos30° + 433·cos60°
= −216.5 + 216.5 = 0 ✓
Verify ΣFy = 0
250·sin30° + 433·sin60° − 500
= 125 + 375 − 500 = 0 ✓
Both positive → our assumed directions were correct.
If either were negative → the cable pulls the other way (but the magnitude is still right).
TA = 250.0 N    TB = 433.0 N
Complete. Every step of the 8-step template used.
